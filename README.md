# 🚀 Student Journal

This project was created purely for fun, amusement, and practicing basic data structures in C. Inside lies a console-based student tracking system packed with unique author features.

## 📦 What's Inside?
Inside is a console **Student Journal** that can add entries, list passing students, and find the top student based on their grade.

### 🛠️ Unique Project Features:
* **Custom User Interface Filter:** A robust user filtering system is baked right in. Try typing `6` in the main menu to see it in action.
* **Dynamic Array (VLA):** The student database size is allocated on the fly based on the class size entered by the user.

## 🚀 How to Run
Just download and launch the pre-compiled **`program.exe`** file on Windows via your command prompt or terminal.
